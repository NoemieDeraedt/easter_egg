Utilisation:

- Ouvrez le fichier 'index.html' placé dans le dossier html.
- Pour faire bouger le personnage, utilisez les flèches de votre clavier ou les touches Z, Q, S et D.
- Pour intéragir avec les livres, utilisez la touche E.
- Pour sortir de l'affichage d'un livre, utilisez la touche Echap de votre clavier.
